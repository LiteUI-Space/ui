## Installation

::CodePreview{title="基本用法" src="CardBasic"}
包含标题、内容、操作区域。
#code
```vue
<script setup lang="ts">
  import { Card } from '@lite-space/ui'
</script>

<template>
  <Card class="w-200px" title="Default size card">
    <template #extra>
      <span>More</span>
    </template>
    <p>Card content</p>
    <p>Card content</p>
    <p>Card content</p>
  </card>
</template>
```
